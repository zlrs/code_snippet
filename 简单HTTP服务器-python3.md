## 在一个目录下启动一个临时的 HTTP 服务
### python2
`python -m SimpleHTTPServer`

### python3
`python3 -m http.server 9000`

## 简单的 http 框架
`pip install simple_http_server`

https://pypi.org/project/simple-http-server/
